# feishu-skill · 飞书笔记整理 + 画板可视化工作流

一个 Claude Code / Claude Agent Skill。读取一份飞书笔记/文档，先把正文里嵌的图片（多为课件截图）用视觉读懂、按类型转成原生可编辑内容——课件图按「知识点→案例」提取成文字、流程图转画板、表格截图转表格、代码截图转代码块，放回原位替换原图，最终不留任何静态图；再把全文整理成结构化的新文档（原文档只读不改、另存一份、文字逐字保真、顺序默认不变），开头新增归类总结 + 总览画板，能可视化的内容用飞书原生可编辑画板（whiteboard）画出来。

## 适用场景

- “帮我整理这篇飞书笔记，能可视化的用画板画出来”
- “把这份文档归纳一下，另存一份新的，别动原文”
- “这篇笔记里有课件图片，先把图里的内容提取成文字再整理”

## 依赖

- `lark-cli`（飞书命令行工具）
- 配套 skill：`lark-shared`（认证/权限）、`lark-doc`（读写文档）、`lark-whiteboard`（画板）

## 安装

把整个目录放到你的 skills 目录下（例如 `~/.claude/skills/feishu-skill/`）即可。

## 文件

- `SKILL.md` — skill 定义与完整工作流（铁律、步骤）。
